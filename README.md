## ✨ Features

- ➕ Add and remove courses dynamically
- 🔢 Weighted GPA calculation on the 4.0 scale
- 🎨 Smooth animations and transitions
- 📱 Fully responsive for mobile
- 👁 Live visit counter
- 🇵🇰 Urdu taglines based on your GPA 😄

| Technology | Usage |

## 🚀 Getting Started

Want to run it locally? It's simple:
```bash
# 1. Clone the repo
git clone https://github.com/your-username/gpa-calculator.git

# 2. Open the folder
cd gpa-calculator

# 3. Open index.html in your browser
# No server needed — just double click index.html!
```
